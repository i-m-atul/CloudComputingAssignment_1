# Django Dockerized Basic App

## Overview

This project is a basic  web application built using Django and Docker. The application serves as a simple demonstration of setting up a Django project within a Docker container and running it.

## Author

- **Roll Number**: G23AI2104

## Application Functionality

The app includes a basic view that returns a "Hello Atul" message. This can be accessed via the `/hello` URL.

## Project Structure

- `Dockerfile`: Specifies the environment setup and dependencies.
- `docker-compose.yml`: Defines the services, volumes, and ports for Docker.
- `requirements.txt`: Lists the Python dependencies for the project.
- `fistopapp/`: The main Django project directory.
- `ecommerceappTomato/`: The Django app directory containing views and URLs.
